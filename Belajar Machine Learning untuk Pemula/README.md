# Belajar Machine Learning untuk Pemula

Ini adalah direktori untuk submission pada kelas **Belajar Machine Learning untuk Pemula** di Dicoding Academy. Proyek ini memuat penyelesaian untuk tugas akhir berupa pembangunan proyek machine learning menggunakan pendekatan *supervised* dan *unsupervised learning*.

## Kriteria yang Telah Dipenuhi
- [x] Memuat Dataset dan Melakukan Exploratory Data Analysis (EDA)
- [x] Pembersihan dan Pra Pemrosesan Data
- [x] Membangun Model Clustering
- [x] Interpretasi Hasil Clustering
- [x] Membangun Model Klasifikasi

## Struktur Direktori
- `submission/` : Berisi source code berupa Jupyter Notebook / Python script serta file pendukung yang dikerjakan pada proyek ini.
- `FEEDBACK.md` : Berisi umpan balik (feedback) dan ulasan lengkap dari reviewer terkait hasil penilaian submission.

Untuk melihat umpan balik dari reviewer Dicoding, silakan buka file [FEEDBACK.md](FEEDBACK.md).
